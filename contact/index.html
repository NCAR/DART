Email us

DAReS staff strive to provide all DART users with an exceptional level of support. Please email us at dart@ucar.edu with questions and suggestions for DART.

File an Issue on GitHub

For issues with software, please file an Issue on GitHub detailing your issue. Since GitHub issues are viewable by the public, filing an issue there will help the larger community by providing searchable documentation of difficulties.
