<a name="OVERVIEW"></a>
<hr>
<h4>OVERVIEW</h4>
<!-- BEGIN OVERVIEW -->
<div>

<tt>platform_mod</tt> is a module that provides public entities whose
value may depend on the operating system and compiler.

</div>
<!-- END OVERVIEW -->
<!-------------------------------------------------------------------->
<a name="DESCRIPTION"></a>
<!-- BEGIN DESCRIPTION -->
<div>

<p>The combination of operating system and compiler is referred to
here as the <i>platform</i>. <tt>platform_mod</tt> provides:<br>

<pre>
  integer, parameter :: r8_kind, r4_kind, &
                        c8_kind, c4_kind, &
                        l8_kind, l4_kind, &
                        i8_kind, i4_kind, i2_kind
</pre>

<p>by use association. These are set to the appropriate KIND values
that provide 4 and 8-byte versions of the fortran intrinsic types. The
actual numerical value of the KIND may differ depending on
platform.</p>

<p>These should be used when the actually bytelength of the
variable is important. To specify numerical precision, Fortran
recommends the use of the intrinsics <tt>SELECTED_REAL_KIND</tt> and
<tt>SELECTED_INT_KIND</tt>.</p>
</div>
<!-- END DESCRIPTION -->
