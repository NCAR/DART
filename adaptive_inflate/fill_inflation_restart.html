Overview

Utility program to fill in the inflation restart file with a fixed inflation and standard deviation value.

For a multi-jobstep run where the only difference between the first step and the rest is whether the inflation is read from the namelist or a restart file, this allows the same namelist to be used in all steps by constructing a restart file for the first job step.

The program reads 2 real values from the console and always writes out a file called "inflate_ics", which can be renamed to match the input inflation restart file in the filter namelist.
